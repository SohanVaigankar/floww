## Tech Stack

- NextJS
- TypeScript
- SASS
- reactflow
- Deployed on vercel

## Run the project

To run the project use clone this repo and navigate to the respective project folder and run the following command:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
